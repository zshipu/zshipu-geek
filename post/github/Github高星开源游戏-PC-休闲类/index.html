Github高星开源游戏 - PC - 休闲类

2022年2月16日
| 阅读
  • AssaultCube - Wouter van Oortmerssen的FPS完全转换，称为Cube。
  • AssaultCube Reloaded - AssaultCube Reloaded。
  • BZFlag - 跨平台，C++的3D坦克FPS。射击，躲闪和夺取旗帜。
  • 城堡 - 黑暗幻想场景中的FPS。包括作为奖励 DOOM E1M1 级别在现代技术中完成的。
  • 欺骗者 - 在这个以世界尽头为背景的哲学射击游戏中，向墙壁，天花板和敌人的头部发射你的蜘蛛无人机。现在就开始玩吧！
  • Doom the Roguelike - DRL（D**m，Roguelike）是一款快速而激烈的咖啡休息Roguelike游戏，其灵感来自流行的FPS游戏Doom by ID Software。
  • dhewm 3 - dhewm 3 是 Doom 3 GPL 源代码端口，知道至少在 Windows、Linux、Mac OS X 和 FreeBSD 上工作。
  • ET遗产 - 沃尔芬斯坦的第二次生命气息：敌人的领地。
  • FreeCS - 反恐精英的自由软件实现。现在就开始玩吧！
  • Freedoom - 基于经典《毁灭战士》引擎的自由软件FPS。
  • GlPortal - 一款现代3D游戏和引擎，带有用C++编写的门户。
  • Hexen II： Hammer of Thyrion - Hammer of Thyrion （uHexen2） 是 Raven Software 的 Hexen II 源代码的跨平台移植版。
  • iortcw - Iortcw是Return To Castle Wolfenstein引擎的港口。
  • ioquake3 - 免费软件FPS引擎，可用于玩雷神之锤3，或制作自己的游戏。
  • LWJake2 - 雷神之锤2的LWJGL端口。
  • openspades - Ace of Spades 0.75的兼容客户端。
  • Red Eclipse - 免费休闲竞技场射击游戏，具有跑酷功能。
  • Smokin’ Guns - 半逼真的模拟”老西部”的伟大氛围。
  • 系统冲击 - 系统冲击（PowerMac版本/官方GPL版本）。
  • 海龟竞技场 - 使用留兰香引擎的第三人称动作游戏。
  • Uebergame - 下一代通用射击游戏。现在就开始玩吧！
  • 虚幻竞技场4 - 虚幻竞技场是Epic和虚幻引擎4开发者社区密切合作创建的。（私人回购）
  • 未被征服 - 快节奏，未来主义的FPS与RTS元素。
  • vkQuake - 基于QuakeSpasm的Vulkan Quake端口。
  • Warsow - 未来卡通世界中的快节奏FPS。
  • Xonotic - Xonotic是一款免费的（GPL）快节奏第一人称射击游戏，可在Windows，OS X和Linux上运行。
