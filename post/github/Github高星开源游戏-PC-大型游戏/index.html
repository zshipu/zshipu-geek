大型多人在线角色扮演

  • 《永恒之地 - 永恒之地》是一款免费的多人在线角色扮演游戏，由Radu Privantu于2002年创建。现在就开始玩吧！
  • FreeSO - 重新實現《The Sims Online》。现在就开始玩吧！
  • 侠盗猎车手 - 侠盗猎车手V的在线合作修改，现在就开始玩吧！
  • Meridian 59 - 第一款3D MMORPG，于1996年发布，并于2012年开源。Meridian 59的原始代码库，更新频率较低。现在就开始玩吧！
  • Multi Theft Auto - Multi Theft Auto是一款开源游戏引擎，为Rockstar North的侠盗猎车手游戏系列添加了完整的多人网络游戏功能，其中最初找不到此功能。设为首页
  • Ryzom - Sci-Fi & Fantasy-MMORPG。现在就开始玩吧！
  • Mana World - 是一款免费的2D开源MMORPG，于2004年发布。
  • TONDE IKO - 一款6屏幕50人HappyFunTimes游戏。
  • 司汤达 - 一个有趣的友好和免费的2D多人在线冒险游戏，具有老派的感觉。现在就开始玩吧！
  • Worldforge - WorldForge是一个免费游戏开发者社区，他们创建了一个名为Dural的虚拟世界系统。

平台

  • Project-AL - Linux Game Jam 2018 的平台游戏。
  • Captain Holetooth - 一款有趣，探索性和故事驱动的平台游戏，带有迷你游戏。用戈多引擎制造。
  • 指挥官天才 - 指挥官基恩兼容克隆。
  • Crazy Battle - owlboy 启发的战斗游戏 - Linux Game Jam 2018 提交。
  • Elemental One - 使用Phaser HTML5游戏框架构建。现在就开始玩吧！
  • 猫的逃脱 - 硬核老派平台游戏取而代之。
  • Frogatto - 动作冒险游戏，由某只堂吉诃德式的青蛙主演。
  • Gish - 屡获殊荣的物理平台游戏。
  • HFT马里奥派对 - 重新混合的本地多人游戏，重新构想了原始马里奥兄弟。
  • 霍克索恩中心之旅 - 这款2D平台游戏基于社区的数字遗产规划剧集，使用LÖVE游戏引擎构建。
  • Juan’s Saga - 一款关于Godot的核心开发人员试图编写新功能并修复错误的传奇的游戏。
  • l’Abbaye des Morts - l’Abbaye des Morts - 移植到Linux，Pandora，GCW0，Wii和PSP。
  • 马里奥-等级-1 - 超级马里奥兄弟的第一关，用Python和Pygame制作。
  • 红色奔跑者 - 红色奔跑者，真棒平台游戏。
  • SuperTux - 经典的2D跳跃和运行侧翻滚带Tux。
  • 埃德加传奇 - 2D平台游戏与持久的世界。
  • Dr. .M的Secret编年史。- 经典风格的2D平台游戏。
  • 狼人 - 你是个狼人，哇！

拼图/卡牌/板

  • 2048-cli - 适用于 Linux 终端的游戏 2048 的 CLI 版本。
  • 2048-in-terminal - CLI版本的游戏2048是正确的。
  • Anonymine - 终端扫雷舰，无需猜测，具有一些独特的功能。
  • ASCIIrinth - 在 ASCII 字符迷宫中找到自己的出路。
  • 光束奥林匹克 - Erlang应用程序，以帮助您检查您的Erlang-Fu。您还可以在本地网络中与朋友一起玩。
  • 蠕变博士的城堡 - 蠕变博士的城堡
  • Dominion - Dominion的C#实现，一个棋盘/纸牌游戏。
  • EbbeFlut - 非常好的棋盘游戏，很久以前就以糟糕的方式实现。
  • Gomoku - Piskworks是一个简单的桌面游戏，称为gomoku。
  • IRMINGARD - 用Ruby编写的耐心纸牌游戏。
  • 激光寺庙 - 与CGA图形的小益智游戏。
  • Lix - Lix：类似旅鼠的游戏，带有谜题，编辑器，多人游戏，现在就玩吧！
  • Minilens - 可爱的益智平台游戏，主演是后世界末日地球上的清洁机器人。
  • Pentobi - 棋盘游戏Blokus的电脑对手。现在就开始玩吧！
  • PokerTH - 德州扑克的实现，支持（在线）多人游戏和机器人游戏。现在就开始玩吧！
  • PuzzleDash - 在Dash引擎中构建的Match-3益智游戏。
  • Quatter - 一款适合两名玩家的棋盘游戏，基于Blaise Müller的Quarto。
  • Rjopek Minigames - Harbour编程语言的多平台游戏。
  • szsol - CLI克隆深圳纸牌游戏。
  • 俄罗斯方块 - 俄罗斯方块Win32，Linux控制台，DOS和ZX Spectrum
  • tictactoe - 图形井字棋AI，永远不会丢失
  • THBattle - 在线Touhou X Sanguosha（Bang的中国变体！）游戏，用纯python实现。论坛
  • Wizznic - 最令人敬畏的益智游戏。

竞速

  • pac-person-freespace - 驾驶和射击游戏，从吃豆人那里获得一些灵感。
