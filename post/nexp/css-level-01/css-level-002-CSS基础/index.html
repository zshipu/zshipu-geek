css-level-002 CSS基础

2022年5月31日
| 阅读

一、 CSS组成部分

    <p>Lorem ipsum dolor sit amet.</p>


    <style>
        p{
            color: red;
        }
    </style>
  1. p : selector 选择器
  2. {}: 放在在大括号内
  3. color: property 属性
  4. red: value 值
  5. ；：每行结尾分号

二、三种方式添加CSS

  1. 外部样式表
    1. 保持.css
    2. link 引入到html
    3. 最常用
  2. 内部样式表
    1. html文件内 style 包裹
    2. 偶热使用
  3. 内联样式
    1. 仅影响一个元素
    2. html元素的style属性中添加
    3. 不推荐使用
