Github高星开源游戏 - 基于浏览器 - 其他类

2022年2月16日
| 阅读
  • 野兽 - 向1984年ASCII游戏”BEAST”致敬，由Dan Baker，Alan Brown，Mark Hamilton和Derrick Shadel用node编写。你必须在可推的方块之间压扁小ASCII野兽，但要注意你不要被它们吃掉。
  • binb - 竞技，多人，实时，猜歌游戏。现在就开始玩吧！
  • CyberPong - 不寻常的乒乓球游戏，可与Makey Makey Kit一起调整。现在就开始玩吧！
  • 糖果盒1 - 基于文本的糖果盒游戏现在玩吧！
  • 糖果盒2 - 基于文本的糖果盒游戏现在玩吧！
  • 牙齿后卫：糖果部落的传奇 - #CandyJam HTML5射击/塔防游戏。现在就开始玩吧！
  • Descensus 2 - TypeScript 2D物理游戏。现在就开始玩吧！
  • 表情妖怪 - 表情妖怪 - 像口袋妖怪一样战斗表情符号！现在就开始玩吧！
  • Game of GitHub - 在任何 GitHub 个人资料贡献图表中播放 Game of Life。
  • Hauberk - Hauberk是一款Roguelike，一款基于ASCII艺术的程序生成的地牢爬行游戏。它是用Dart编写的，并在您的浏览器中运行。
  • HeartLine - 48 小时为 2013 年全球游戏开发大赛制作的游戏。
  • KVH Tabu - 基于浏览器的类似Tabu的游戏。现在就开始玩吧！
  • Massive Decks - Massive Decks是一款基于优秀的反人类卡牌的网页游戏 - 一个派对游戏，您可以在其中玩白色卡牌，以尝试在与给定的黑卡结合使用时产生最有趣的结果。现在就开始玩吧！
  • 需要大量组装 - 在基于网格的多人游戏中对机器人的类似 8086 的微处理器进行编程。现在就开始玩吧！
  • Ogar - 用 NodeJS 编写的 Agar.io 服务器的开源实现。
  • Psiral - JavaScript/HTML5 游戏，GitHub Game Off 2 获胜者。现在就开始玩吧！
  • Squirts - 用JavaScript实现的知名独立游戏。现在就开始玩吧！
  • 杀手 - 基于Flash/ActionScript3的”非游戏”。现在就开始玩吧！
